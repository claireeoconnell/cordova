external_auth()

external_auth( $prefix) : void

External Auth

Calls on a model to handle external authentication. Useful for cases such as logging in via University servers, LDAP, Google authentication, etc.

Place your external authentication model in application/models/external_auth/PREFIX_auth_model.php For example, if you are using Google authentication, then put it in application/models/external_auth/google_auth_model.php

In your model create a function called authenticate(), which will be called from here to begin the external authentication process.

The model should be in charge of: 1) Authenticating the user via external means 2) Logging in the user if they're registered to use the site 3) Error handling (incorrect username/password, non-registered user, etc.)
